# Phase 03 Plan: Structured Data Enhancement

**Plan ID:** 03-01
**Created:** 2026-04-03
**Wave:** 1

---

## Objective

为所有页面添加完整的 Schema.org 结构化数据，最大化富媒体搜索结果展示。

---

## Requirements (from ROADMAP)

| ID | Description |
|----|-------------|
| SCHEMA-01 | Scene landing pages have FAQPage schema |
| SCHEMA-02 | Blog articles have Article schema |
| SCHEMA-03 | All pages have BreadcrumbList schema |
| SCHEMA-04 | Product pages have expanded FAQ sections with FAQPage schema |
| SCHEMA-05 | All schema passes Google Rich Results Test |

---

## Tasks

### Task 1: Audit Current Schema Status

Check all pages for existing schema:
- 5 product pages
- 5 blog articles
- 10 AI Photo landing pages
- Main pages (index, about, blog, help, faq)

### Task 2: Add BreadcrumbList Schema to All Pages

Create breadcrumb schema template and add to all pages.

### Task 3: Verify/Enhance FAQPage Schema

- AI Photo landing pages (10) - verify FAQPage
- Product pages (5) - add/expand FAQPage
- Blog articles (5) - verify FAQPage (added in Phase 2)

### Task 4: Verify/Enhance Article Schema

- Blog articles (5) - verify Article schema
- Add author, publisher, date info

### Task 5: Rich Results Validation

Run all pages through Google Rich Results Test and fix errors.

---

## Success Criteria

1. ✅ All pages have BreadcrumbList schema
2. ✅ All landing pages have FAQPage schema
3. ✅ All blog articles have Article + FAQPage schema
4. ✅ All product pages have expanded FAQPage schema
5. ✅ All schema passes Rich Results Test
