## 二、你在 Product Hunt 上到底要提交什么

你今天要提交的是一个**产品页面**，包含这些内容：

1. Product Name（产品名）
2. Tagline（一句话卖点）
3. Description / Short Description（简短介绍）
4. Website（官网链接）
5. Play Store / Download Link（下载链接）
6. Thumbnail / Gallery（封面图、截图）
7. Maker 信息（提交人资料）
8. Maker Comment（产品上线后的第一条官方评论）

所以结论很直接：

- **要提交产品**
- **不是只写 comment 就算完成**
- comment 是产品提交后的配套动作

## 五、一步一步怎么操作

### 步骤 1：打开提交入口
进入 Product Hunt 的提交入口，然后开始创建产品。

你要理解成：
- 不是“发帖”
- 是“创建一个产品 listing”

### 步骤 2：填写产品基础信息
把下面这些字段填进去：
- Product Name → `Thermometer - Hygrometer`
- Tagline → 上面的英文 tagline
- Short Description / Description → 上面的英文 description
- Website → 官网链接
- Play Store / URL → Google Play 链接

### 步骤 3：上传封面图
最少传 1 张。

如果可以继续补 gallery：
- 再传 3-5 张截图
- 优先展示：温度、湿度、气压、钓鱼/天气场景

### 步骤 4：补充 Maker 信息
如果系统要求补资料：
- 只填写必要项
- 填完继续，不要停留太久

### 步骤 5：保存 / 提交
Product Hunt 可能出现几种状态：
- Draft
- Submitted
- In Review
- Scheduled

今天你的最小目标不是“爆发”，而是：
- **至少把 draft 建起来**，或者
- **正式提交进入 review / scheduled**

### 步骤 6：提交后写 Maker Comment
产品页建立后，补第一条官方评论。

这条 comment 的作用：
- 解释为什么做这个产品
- 说明差异化
- 给访问者一个可信的第一印象

```text
Hi Product Hunt! 👋

We built Thermometer - Hygrometer because we noticed a common misconception: many "thermometer apps" claim to measure room temperature using phone sensors—but those sensors only measure phone internal temperature (CPU/battery heat), not ambient temperature.

Our approach is different: we use GPS positioning to fetch accurate local weather data from nearby stations. This gives real ambient temperature, humidity, and barometric pressure—perfect for fishing enthusiasts (barometer predicts fish activity!), health monitoring, and weather planning.

The app has grown to 1M+ downloads organically, with users praising the barometer feature for fishing trips.

Happy to answer questions about weather apps, fishing tips, or building privacy-first tools!
```

## 八、提交日建议

原文档中的建议是：
- 最佳日期：Tuesday-Thursday
- 最佳时间：8-9 AM Pacific Time
- 避开周末和节假日

但对你现在来说，底层逻辑不是“卡最优时间”，而是：
- **先把第一条链路跑通**
- **先把产品页建立起来**
- **先拿到 draft / submission 状态**

所以：
- 如果今天就能提，先提
- 不要因为等“最佳时间”继续拖延

## 九、提交后要做什么

### Submission Day
产品提交流程完成后：
1. 写第一条 maker story / maker comment
2. 及时回复评论
3. 感谢 upvoters
4. 回答问题时保持透明

### After Launch
如果后续正式上线：
1. Share on social media
2. Update website with Product Hunt badge
3. Collect feedback for improvements
