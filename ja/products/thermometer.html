Thermometerアプリ

AndroidでThermometerをローカルの天気ベース参照として活用し、温度、湿度、気圧、快適条件をチェック。毎日のプランニング、家庭の快適さチェック、屋外コンディションの把握に役立ちます。

読み取り値はお使いのデバイスの位置と付近の天気データに基づいています。デバイス固有の読み取り値が必要な場合は、専用の室内、スマートホーム、医療、またはプロの気象計器をご使用ください。

アプリスクリーンショット

Thermometerがローカルの温度、湿度、気圧情報をどのように表示するかをご覧ください

主な機能

温度、湿度、気圧の参照に役立つ天気ベースツール

🌡️

温度参照

GPS位置と付近の天気データを使用してローカルの温度をチェック。日常のプランニングや素早いコンディション確認のために、摂氏または華氏で読み取り値を表示。

  • ✓ 位置ベースの読み取り
  • ✓ 摂氏＆華氏
  • ✓ クイックリファレンス
💧

湿度表示

温度や気圧とともにローカルの湿度を確認し、快適さ、換気、周囲のコンディションの変化をより適切に判断。

  • ✓ 湿度パーセンテージ
  • ✓ 快適さコンテキスト
  • ✓ 統合天気ビュー
📊

気圧トレンド

屋外プランニング、天気認識、その他のコンディションベースの活動のために、気圧の読み取り値とトレンドの変化を確認。

  • ✓ hPa/mbarで気圧表示
  • ✓ トレンドインジケーター
  • ✓ 屋外プランニング参照
😊

不快指数

温度と湿度に基づいたシンプルな快適さの参照を取得し、コンディションが快適、蒸し暑い、暖かいのいずれかを素早く判断。

  • ✓ 快適スコア
  • ✓ 日次プランニング補助
  • ✓ 読みやすいサマリー

よくある質問

Thermometerアプリに関するよくあるご質問

Thermometer - Hygrometerとは何ですか？

Thermometer - HygrometerはGPSベースの天気データを使用してローカルの温度、湿度、気圧を表示するAndroid天気アプリです。利用可能な読み取り値はお使いのデバイスの位置接続付近のデータソースによって異なります。

Thermometerアプリの精度はどのくらいですか？

アプリはGPS測位を使用して付近の天気データを取得します。精度はお使いのデバイスの位置信号、インターネット接続、付近の気象ステーションデータの品質によって異なります。

室内温度計の代わりになりますか？

いいえ。ThermometerはGPS位置と付近の天気データを使用してAndroid上で一般的な環境参照を提供します。デバイス固有の読み取り値が必要な場合は、専用の室内、スマートホーム、医療、またはプロの気象計器をご使用ください。

Thermometerアプリは無料ですか？

はい、Thermometer - Hygrometerは広告付きで無料でダウンロードできます。拡張予報や位置履歴などのプレミアム機能にはオプションのアプリ内課金が利用可能です。

アプリはオフラインで動作しますか？

天気データを更新するにはインターネット接続が必要で、GPSは位置ベースの読み取りに使用されます。デバイスがオフラインの場合、利用可能性が異なる場合があります。

対応するAndroidバージョンは？

ThermometerアプリはAndroid 5.0（Lollipop）以降が必要です。

不快指数とは何ですか？

不快指数は温度と湿度に基づいたシンプルな快適さの参照を提供し、日常のプランニングや服装の決定に役立ちます。

釣りに使えますか？

釣り旅行を計画する際の参照として気圧の傾向を使う方もいます。アプリは気圧の変化を確認できますが、釣りのコンディションは天気、場所、季節、その他多くの要因にも左右されます。
